# FilmTrace - 胶片摄影参数记录 App 规格文档

## 1. 项目概述

**项目名称**: FilmTrace
**项目类型**: Android 原生应用 (Kotlin)
**核心功能**: 快速、方便地记录胶片摄影时的各项参数，支持胶卷管理、拍摄记录、参数预设和历史查看。

## 3. 功能列表

### 3.1 胶卷管理
- 添加/编辑/删除胶卷档案（品牌、型号、ISO、画幅 135/120/220/4x5）
- 标记胶卷状态：已装载 / 库存 / 已拍完
- 显示剩余张数

### 3.2 拍摄记录
- 为当前胶卷新增一条拍摄记录
- 记录参数：
  - **快门速度**（1s, 1/2, 1/4, 1/8, 1/15, 1/30, 1/60, 1/125, 1/250, 1/500, 1/1000, 1/2000, 1/4000, B门）
  - **光圈**（f/1.0 ~ f/22，标准整档）
  - **ISO**（自动继承胶卷 ISO，支持手动微调 ±1/3 ~ ±3）
  - **镜头焦距**（手动输入，单位 mm）
  - **对焦距离**（可选，近摄/中距/远距 或 具体米数）
  - **曝光补偿**（±3 EV，1/3 档位）
  - **滤镜**（UV/偏振/渐变灰/中性密度/彩色滤镜）
  - **场景描述**（简短备注，如"逆光人像"）
- 时间戳自动记录

### 3.3 参数预设（曝光组合）
- 保存常用的快门+光圈组合为预设（如"晴天 f/8 法则"对应的 1/125@f/8）
- 一键应用到当前拍摄记录

### 3.4 历史与浏览
- 按胶卷查看所有拍摄记录
- 按日期浏览全部记录
- 搜索/筛选功能
- 查看单张记录的详细信息

### 3.5 数据导出
- 导出为 CSV 格式，方便在表格软件中整理或打印

## 4. UI/UX 设计方向

### 4.1 整体视觉风格
- **Material Design 3**（Material You）
- 深色主题为主，符合胶片摄影的复古/暗房氛围
- 浅色主题可选
- 强调排版层次，信息一目了然

### 4.2 配色方案
- **深色主题主色**: 暖琥珀色 (#D4A574) —— 致敬胶卷轴和暗房安全灯
- **深色背景**: #1A1A1A
- **表面色**: #252525
- **强调色**: 暖橙 (#E8956A)
- **文字**: 高对比度白色 (#F5F5F5) / 低对比度灰色 (#9E9E9E)

### 4.3 布局方式
- **底部导航栏**（3 个主要标签）：
  1. **拍摄** — 当前胶卷 + 快速记录入口
  2. **胶卷** — 胶卷库列表管理
  3. **历史** — 全部拍摄记录浏览
- 浮动操作按钮（FAB）用于新建记录
- 拍摄记录页：分段式表单，大量使用滚轮/选择器交互，减少打字

### 4.4 交互设计
- 拍摄页面支持**大按钮快速输入**（快门/光圈各占屏幕一半，一键点选）
- 震动反馈（Haptic）确认操作
- 防止误操作：删除需二次确认
